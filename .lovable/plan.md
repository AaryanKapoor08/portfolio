

# Aaryan Kapoor — Portfolio Website

## Design
- **Style**: Light & airy — white/cream background, soft shadows, refined typography
- **Font**: Clean sans-serif (Inter for body, a serif or display font for headings for contrast)
- **Accent color**: Subtle warm or cool tone (soft blue or muted charcoal)
- **Animations**: Smooth scroll-reveal on each section using Framer Motion fade/slide-up

## Layout — Single Scrollable Page

### 1. Hero
- Name, title ("Full Stack AI Developer"), one-liner tagline
- Subtle animated gradient or soft background texture
- CTA buttons: "View Projects" (scrolls down) + "Get in Touch"
- Links to email, LinkedIn, GitHub

### 2. Projects (Main Focus — 4 detailed cards)
Each project gets a rich card with:
- Project name, tech stack badges, one-line description
- Key highlights (2-3 bullet points)
- Live demo link + GitHub link
- Projects: **Auctus AI**, **Prompt Amplifier**, **RAG Pipeline**, **Hangry**

### 3. About / Skills
- Brief bio paragraph
- Skills grouped visually: Languages, AI/ML, Web, DevOps, Tools
- Clean pill/badge layout

### 4. Experience & Certifications
- Timeline-style layout for experience (Ideation Boost Camp, Code Social)
- Certifications listed as clean cards (Oracle GenAI, APEX, HackFest)

### 5. Education
- University of New Brunswick — CS, GPA 3.9/4.2

### 6. Contact / Footer
- Email, LinkedIn
- Simple contact form or mailto CTA
- Minimal footer

## Interactions
- Sticky top nav with smooth scroll to sections
- Sections animate in on scroll (fade + slide up)
- Hover effects on project cards and buttons
- Mobile responsive throughout
